# Harry Potter Book of Spells

## For students, Witches and Wizards alike

### Features: 
* Houses
* Spells
* Elixirs
* Ingredients
* Notes (requires a user account)

### To launch app: 

Optimised for viewing on iPhone 13 Pro - iOS 15.5

```bash
cd HPSpellBook
```
```bash 
npm install
```
```bash 
expo start
```
